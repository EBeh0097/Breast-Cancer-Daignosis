# Breast-Cancer-Daignosis

<h1>Data Mining - Predicting malignant (cancerous) vs benign (non-cancerous) </h1>


<h2>Description</h2>
To classify a tumor as either malignant (cancerous) or benign (non-cancerous) based on various features computed from a digitized image of a fine needle aspirate (FNA) of a breast mass. The dataset contains 569 samples (instances) and 30 real-valued, positive, numerical features for each. There are two target classes: Malignant (212 sample) and Benign (357 samples).
<br />
